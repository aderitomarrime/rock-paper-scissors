## Overview
A simple Rock, Paper and Scissors web game where the user plays against the computer. The first to reach 5 points wins.


## Live Demo

Visit the website: [Live Demo](https://aderitomarrime.github.io/rock-paper-scissors/)


## Features

- Minimal interface 
- Score tracking during the session
- Random computer moves for fair gameplay
- Instant win/lose/draw feedback


## Technologies Used

- HTML5  
- CSS3 
- JavaScript


## Installation

To run this project locally, follow these steps:

1. Clone the repository:

```sh
git clone https://github.com/aderitomarrime/rock-paper-scissors.git
```

2. Navigate to the project directory:

```sh
cd rock-paper-scissors
```
3. Open index.html in your browser


## How to use

Once the website is open, click Rock, Paper, or Scissors to start a round. You will immediately see the result of that round.  
Keep playing until you or the computer reaches 5 points to win. Once either you or the computer reaches 5 points, clicking again will automatically restart the game.
